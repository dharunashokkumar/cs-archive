CTF platforms, hacking labs, security tools, and ethical hacking guides.

Content Index

  • HackTheBox (CTF Platform) - A massive hacking playground for practicing penetration testing skills.
  • TryHackMe (Training) - Hands-on cyber security training through guided rooms.
  • OverTheWire (Wargames) - Learn security concepts by solving wargames like Bandit and Natas.
  • Burp Suite (Tools) - The de facto standard tool for web application security testing.
  • Wireshark (Tools) - The world's foremost network protocol analyzer.
  • Metasploit (Tools) - The world's most used penetration testing framework.
  • OWASP (Resources) - The Open Web Application Security Project. Home of the Top 10.
  • GTFOBins (Resources) - Curated list of Unix binaries that can be used to bypass local security restrictions.
  • Ghidra (Tools) - A software reverse engineering (SRE) suite of tools developed by NSA.
  • PicoCTF (CTF Platform) - A free computer security education program with original content.
  • Nmap (Tools) - Free and open source utility for network discovery and security auditing.
  • Kali Linux (OS) - Advanced Penetration Testing Linux Distribution.
  • Parrot OS (OS) - The ultimate framework for your cyber security operations.
  • Hashcat (Tools) - World's fastest and most advanced password recovery utility.
  • John the Ripper (Tools) - Open Source password security auditing and password recovery.
  • Aircrack-ng (Tools) - Complete suite of tools to assess WiFi network security.
  • OWASP ZAP (Tools) - The world's most widely used web app scanner.
  • Shodan (Search Engine) - Search engine for Internet-connected devices.
  • Censys (Search Engine) - Search for devices, networks, and infrastructure on the internet.
  • VirusTotal (Resources) - Analyze suspicious files and URLs to detect malware.
  • Fail2Ban (Tools) - Intrusion prevention software framework that protects computer servers from brute-force attacks.
  • Snort (Tools) - Open Source Intrusion Prevention System (IPS).
  • Suricata (Tools) - High performance Network IDS, IPS and Network Security Monitoring engine.
  • Wazuh (Platform) - The Open Source Security Platform. XDR and SIEM.
  • OpenVAS (Tools) - Full-featured vulnerability scanner.
  • Nessus Essentials (Tools) - The #1 vulnerability assessment solution.
  • Sqlmap (Tools) - Automatic SQL injection and database takeover tool.
  • Nikto (Tools) - Web server scanner which performs comprehensive tests.
  • Gobuster (Tools) - Directory/File, DNS and VHost busting tool written in Go.
  • Dirb (Tools) - Web Content Scanner. It looks for existing (and/or hidden) Web Objects.
  • Amass (Tools) - In-depth Attack Surface Mapping and Asset Discovery.
  • Maltego (Tools) - Graph link analysis tool for gathering and connecting information.
  • Recon-ng (Tools) - Full-featured Web Reconnaissance framework written in Python.
  • Social Engineering Toolkit (Tools) - Penetration testing framework designed for social engineering.
  • BeEF (Tools) - The Browser Exploitation Framework.
  • Hydra (Tools) - Parallelized login cracker which supports numerous protocols.
  • Medusa (Tools) - Speedy, massively parallel, modular, login brute-forcer.
  • Mimikatz (Tools) - A little tool to play with Windows security.
  • Empire (Framework) - Post-exploitation toolkit.
  • BloodHound (Tools) - Six Degrees of Domain Admin. Active Directory trust relationship analysis.
  • Impacket (Library) - Collection of Python classes for working with network protocols.
  • Responder (Tools) - LLMNR, NBT-NS and MDNS poisoner.
  • Mirekusoft Install Monitor (Tools) - Monitor and analyze application installations.
  • Process Hacker (Tools) - A free, powerful, multi-purpose tool that helps you monitor system resources.
  • Sysinternals Suite (Tools) - The definitive suite of tools for Windows troubleshooting.
  • CyberChef (Tools) - The Cyber Swiss Army Knife - a web app for encryption, encoding, compression and data analysis.
  • PortSwigger Academy (Training) - Free web security training from the creators of Burp Suite.
  • PentesterLab (Training) - The best place to learn Web Penetration Testing.
  • VulnHub (Labs) - Provides materials that allow anyone to gain practical 'hands-on' experience.
  • Exploit Database (Resources) - The ultimate archive of Exploits, Shellcode, and Security Papers.
