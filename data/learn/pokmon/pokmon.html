Development Projects

APIs

  • PokeAPI - A RESTful API for Pokémon (http://pokeapi.co/).
  • pgoapi - Pokémon GO API lib/demo.
  • pokemon-API - Beginner friendly API endpoint that returns Pokémon Data in JSON format (https://app.pokemon-api.xyz).
  • Pokemon-GO-node-api - Pokémon GO API Node.js library.
  • Pokemon-Go-Rocket-API - Client API library for Pokémon GO in C#.
  • Pokemon GO Java API - Pokémon GO API for Java.
  • Pokemon Name API - Get a list of Pokémon names in JSON.
  • PokemonGO-Pokedex - Pokédex of Pokémon GO in JSON.
  • PokeTypes - Get Pokémon types, weaknesses and strengths.
  • TCGdex - Multi languages Pokémon TCG API.
  • Kotlin-Pokedex - Pokedex app built with Kotlin.
  • graphql-pokemon - Get information of a Pokémon with GraphQL.

Apps

  • Pokemon-Showdown - Pokémon battle simulator.
  • iPokeMon - Pokémon like game on iOS with Location Based Service.
  • pokemon-online - Online Pokémon Battle Simulator.
  • Shuffle-Move - A program to identify and display the best next move for the game Pokémon Shuffle.
  • pokecry - Guess Pokémon from their cry.
  • Pokedex - Android Pokedex using Dagger, Hilt, Motion, Coroutines, Flow, and Jetpack.
  • Who's That Pokémon? - Guess Pokémon from their silhouette.

Bots

  • Pokemon Go Bot - Pokémon GO bot that farms for Pokestops and Pokémon.
  • slack-pokemon - A bot for having Pokémon battles in Slack.
  • PokeSlack - Alerts you of nearby Pokémon.
  • TwitchPlaysPokemon - A bot mimicking Twitch Plays Pokémon.
  • PokemonGo-Bot - The Pokémon GO bot script, baking with community.
  • Poketrainer - A fast and feature filled bot in Python.
  • PokeBot - The automated Pokémon speedrunning machine.
  • PokemonGoBot - Bot that plays Pokémon GO.
  • Bot Builder for PokemonGoBot - Simply build your Pokémon GO bot.
  • PokeBuddy - A Pokémon GO bot.

Miscellaneous

  • alfred-pokedex - Alfred 3 workflow to lookup information about Pokémon.
  • hyper-pokemon - Wondrous, tailor-made Pokémon themes for your Hyper terminal.
  • Pokemon-Go-Controller - Play Pokémon GO safely or at unavailable area.
  • pokecrystal - Disassembly of Pokémon Crystal.
  • x-pokemon - A web component to display Pokémon.
  • pokered - Disassembly of Pokémon Red/Blue.
  • pokemon-reverse-engineering-tools - Tools for compiling and disassembling Pokémon Red and Pokémon Crystal.
