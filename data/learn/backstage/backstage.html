Plugins

A collection of awesome Backstage plugins to enhance your platform's capabilities.

Core Plugins

  • Catalog - Provides an inventory of all components in your ecosystem.
  • TechDocs - Manage and display technical documentation for your components.
  • Scaffolder - Create new components using templates.
  • Cost Insights - Gain insights into your cloud costs.

Community Plugins

  • Community Plugins - Where Backstage maintains the list of community plugins available.
  • GitHub Actions - Monitor your GitHub Actions workflows.
  • Jenkins - Integrate Jenkins for CI/CD.
  • SonarQube - Analyze and monitor code quality with SonarQube.
  • CircleCI - Monitor your CircleCI builds.
  • Prometheus - Monitor application performance using Prometheus.
  • Bazaar - Discover and collaborate on internal projects within your organization.
  • backstage-aws-cloudformation-plugin - Manage AWS CloudFormation stacks.
  • aws-lambda - Manage and monitor your AWS Lambda functions.
  • AiKA - AI-powered assistant for context-aware answers from your internal docs and APIs.
  • RAG AI Plugin - Adds retrieval-augmented generation using LLMs for your internal knowledge.
  • Announcements Plugin - Create and manage internal announcements directly in Backstage.
  • Git Release Manager - Automate GitHub release workflows from within Backstage.
  • Port Plugin - Sync data from across your stack into scorecards and dashboards.
  • Env0 Plugin - Manage Infrastructure as Code (IaC) workflows and self-service provisioning.
  • Tech Insights Scorecards - Automate service maturity scorecards with centralized checks across your catalog.
  • Scaffolder Yeoman Module - Extend golden paths with Yeoman blueprints and new template actions like GitHub Issues or Terraform Cloud.
  • Microsoft Graph Catalog Module - Synchronize Azure AD users and groups as Backstage entities.
  • Azure DevOps Catalog Modules - Bring Azure Repos, Boards, and Pipelines metadata into the catalog backend.
  • Search Backend Modules - Unlock faster, filtered search with Elasticsearch or Lunr-powered hybrid indexing.
  • GitHub Projects - Manage and triage GitHub Projects (v2) without leaving Backstage.
  • New Relic - Jump straight from service catalog entries to New Relic dashboards.
