Awesome Refinery CMS

A collection of awesome Refinery CMS extensions, resources and shiny things.

Frontoffice extensions

  • refinerycms-copywriting - Write once, display at the infinite
  • refinerycms-elasticsearch - Fulltext search with Elasticsearch
  • refinerycms-image-slideshows - Display slideshows
  • refinerycms-jobs - Job Posting and Job Applications
  • refinerycms-products - Display products
  • refinerycms-retailers - Display retailers
  • refinerycms-teams - Display teams
  • refinerycms-testimonials - Manage and display testimonials
  • refinerycms-videos - Basic display of Youtube videos

Backoffice extensions

  • refinerycms-dynamicfields - Create models dynamically and attach them to pages
  • refinerycms-page-resources - Attach resources (files) to a Page
  • refinerycms-redactor - WYSIWYG editor with Redactor
  • refinerycms-resource-categories - Categorize resources
  • refinerycms-snoods - Creating download pages and sending leads to Salesforce
  • refinerycms-tinymce - TinyMCE Visual Editing

Solidus e-commerce + Refinery extensions

  • solidus-refinery-authentication - Shared auth between Solidus and Refinery

Spree e-commerce + Refinery extensions

  • spree_refinery_resources - Attach resources (files) to a Spree::Product
  • spree-refinery-authentication - Shared auth between Spree and Refinery
