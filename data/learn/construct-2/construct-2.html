Hosting

  • itch.io - Platform where to host, showcase, promote or sell your games - Free.
  • GameJolt - Similar platform to itch.io, with a focus on social communities and gamification - Free.
  • GitHub Pages - Platform where to host an exported game as a static website - Free.
  • Surge - Platform where to host an exported game as a static website - Free/Paid.
  • Vercel - Platform where to host an exported game as a static website or cloud functions - Free/Paid.
  • Netlify - Platform where to host an exported game as a static website - Free/Paid.

Export

Construct 3

  • Chowdren - Blazingly fast runtime for Clickteam Fusion and Scirra Construct.
  • Pipelab - Fully automated export pipeline.

Construct 2

  • AppPlayer-Electron - Preview apps/games in an Electron prebuilt exe - as if Electron were a normal browser! Currently Windows only, but soon™ for Mac and Linux.
  • Fresh NW - Easily setup exporting for the latest versions of NW.js, replacing the outdated "NW.js for Construct 2" installer.
  • website Reinarte - Plugins, Behaviors and examples.
